# 🧮 Ultimate Utility Calculator

A simple multi-functional calculator web application built using HTML, CSS, and JavaScript.  
It includes a basic calculator, scientific functions, unit converter, and multiple themes in one clean interface.

---

## 🚀 Features

### 🧮 Basic Calculator
- Addition (+), Subtraction (−), Multiplication (×), Division (÷)
- Decimal support
- Backspace and clear (C)
- Calculation history

---

### 🔬 Scientific Calculator
- sin, cos, tan
- log (base 10)
- square root (√)
- power functions (x², x³, xʸ)
- constants: π and e

---

### 📏 Unit Converter
- Kilogram ↔ Gram
- Meter ↔ Kilometer
- Simple unit conversion system (expandable)

---

### 🎨 Themes
- Light Mode ☀️
- Dark Mode 🌙
- Ocean Theme 🌊
- Neon Theme ⚡

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

---

## 📁 Project Structure
- index.html → UI structure
- style.css → Styling and themes
- script.js → Calculator logic
